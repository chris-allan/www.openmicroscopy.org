OMERO Apps

Extend the functionality of OMERO with web or command-line apps

Web apps

OMERO.web apps add new tools that can be accessed from the OMERO.webclient or embedded in the webclient itself.
Apps must be installed by the sysadmin on the OMERO.web server.

OMERO.iviewer

A multi-dimensional image viewer with support for tile-based images, ROI editing and side-by-side image comparisons.

OMERO.iviewer

OMERO.figure

A tool for quickly creating high-quality figures without needing to export images from OMERO.

OMERO.figure

OMERO.parade

Filter and plot images by analysis results or other metadata.

OMERO.parade

OMERO.fpbioimage

A volumetric web-based image viewer.

OMERO.fpbioimage

OMERO.mapr

Browse your images via Key-Value pairs.

OMERO.mapr

OMERO.gallery

Filter and plot images by analysis results or other metadata.

OMERO.gallery

Command-line plugins

The OMERO command line is a powerful tool that can be extended with plugins to perform a range of tasks.

OMERO-cli-render

Update rendering settings on a large number of images.

omero-cli-render

OMERO-metadata

Convert tabular data from CSV files to OMERO.tables.

omero-metadata
